1. 没有类似 js 字面的方式返回一个对象，一般都是先定义一个 struct
2. 一般的实现都是 impl 一个 trait 或者结构体作为静态方法，而不是纯函数（工具函数除外）
3. Rust 里面没有 TS 的联合类型一样的特性，可以用枚举来替代
4. 想要实现类似 js 里面判断类型然后调用不同的函数的行为，可以用 trait 来替代，原理就是同一个类有同名的函数，但是是不同 struct 的实现。
5. rust 里面没有像 TS 一样的 Obj['xx'] 取类型，只能提前声明一个别名（type xx = xx），然后复用类型
6. Rust 里一般索引、计数（count）等跟内存大小、集合长度相关的数值，通常都用 `usize` 类型，如果是固定的数值则可以不用
